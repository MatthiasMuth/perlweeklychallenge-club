JSON example data for Perl Weekly Challenges

This site provides JSON example data for Perl Weekly Challenges.
New challenges are usually added shortly after publication, which should make the data useful both for the current week's challenge and for exploring older tasks.

The site also includes Perl skeleton scripts for each challenge task. Each skeleton contains

  • a subroutine stub, whose name and parameters are derived from the challenge task title and example data,
  • boilerplate code that reads the corresponding JSON file and runs tests for all published examples.
    (The boilerplate code is the same for all challenge tasks, no need for any adaptions.)

The idea is that you can start working on the solutions right away after downloading these files, and immediately test whether your solutions work for the published examples, just by running the Perl script.

Download or repository access:

Files can be downloaded directly, e.g. 

curl -O https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-1.json
curl -O https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-1.pl
curl -O https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-2.json
curl -O https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-2.pl
or, more simply if you use bash or a similar shell: 
curl --remote-name-all \
    https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-{1,2}.{json,pl}

All files are also available in Git in the json-examples branch in my repository. https://github.com/MatthiasMuth/perlweeklychallenge-club/tree/json-examples https://github.com/MatthiasMuth//json-examples/challenge-378
