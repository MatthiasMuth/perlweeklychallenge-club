# Perl Weekly Challenge JSON Example Data

This branch contains Perl Weekly Challenge tasks example data as JSON files,
and skeletons for Perl solutions
that include boilerplate code
for testing the solution using those JSON files.<br/>
The idea is that you can start working on the solutions right away
after downloading these files.

### Downloading and repository access
Files can be downloaded directly from GitHub Pages, e.g.

```bash
curl -O \
https://matthiasmuth.github.io/perlweeklychallenge-club/examples/challenge-378/ch-1.json
```

All these files are also available in Git
in the [`json-examples`]( 
https://github.com/MatthiasMuth/perlweeklychallenge-club/tree/json-examples)
branch in my repository.
